# healthcare-sql-analysis

# Project Overview

This project analyzes hospital patient data using PostgreSQL. The dataset includes patient identiifiers, medical conditions, admissions, billing, and medications.

# Tools Used
PostgreSQL (pgAdmin)
SQL (Data cleaning and analysis)
Excel (initial data cleaning)

# Key Analysis Performed
Total patient count
Most common medical conditions
Medication usage trends
Admission type distribution
Average billing per condition
Monthly admission trends

# Skills Demonstrated
SQL data cleaning (ALTER TABLE, NULL handling)
Aggregations (COUNT, AVG)
GROUP BY analysis
Data validation
